## gitbook 文档工具
本工具为gitbook常用功能组件组合，支持快速的编写markdown，生产HTML文档。

## 操作步骤
1. 安装node和npm

2. 安装gitbook 和常用组件
``` js
/**全局安装gitbook  **/
npm install gitbook-cli -g

/**安装gitbook 常用组件(book.json配置)  **/
gitbook install

```
3. 预览和打包生产html

``` js
/**本地预览book **/
gitbook serve

/**打包生产html  **/
gitbook build

```
